<!-- usage -->
```sh-session
$ npm install -g @xterr/nft-generator
$ nft-generator COMMAND

$ nft-generator (-v|--version|version)

$ nft-generator --help [COMMAND]
USAGE
  $ nft-generator COMMAND
...
```
